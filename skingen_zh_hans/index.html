真正的难点

可玩的 Minecraft 角色，是一个穿着标准 2D UV 皮肤的 3D 方块身体

Minecraft 按固定 UV 坐标读取皮肤

身体区域：头、身体、手臂和腿都有固定位置。
外层区域：第二层纹理用于头发、帽子、袖子和盔甲。
Alpha：透明外层像素会消失，不透明像素会变成凸起细节。

Minecraft 会把基础层和外层渲染成分层几何

内层：基础身体纹理。
外层：稍大的几何体，用来表现头发、帽子、外套、盔甲和凸起细节。
头部外层方块大约凸出半个像素。
身体外层使用不同缩放。
手臂 / 腿凸起细节跟随各自部位的几何。
2D UV 布局 → 3D 方块身体
先看 UV 图，再看内外层，最后看这张平面图如何包到 3D 身体上。
为什么容易出错

AI 必须同时遵守三条 Minecraft 皮肤规则

1

像素位置

一个区域错位，就可能把手臂、脸或衣服细节贴到错误部位。

2

UV 到 3D 几何

头发、外套和围巾必须跨过不同身体区域后仍然连续。

3

Alpha 可见性

外层需要透明信息，但大多数图像模型输出的是普通 RGB 图片。

这不是风格滤镜，而是一个受约束的 2D 到 3D 纹理编码问题。

后处理和 Alpha Marker

从 RGB 生成结果中恢复透明信息

扩散模型通常输出 RGB 图片，而不是干净的 RGBA 纹理。
灰色背景阈值可能把前景像素和背景像素混在一起。
Alpha Marker 会在透明 UV 像素中心画一个很小的白色锚点。
后处理检测这些锚点，并把对应像素分类为透明。
Alpha Marker 把模糊的前景分离变成特征检测。
